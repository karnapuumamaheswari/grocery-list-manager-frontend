# Grocery List Manager - Frontend

## Project Title
Smart Grocery List Manager (Frontend)

## Project Description
A full-stack grocery management frontend built with React. It helps users manage grocery lists, pantry inventory, budget tracking, recipe suggestions, and purchase history using a clean dashboard experience.

## Features
- Smart grocery list creation and editing
- Current list view with share option
- Pantry tracking with low-stock and expiry alerts
- Pantry-first recipe suggestions with missing-ingredient add flow
- Budget planner with projected monthly spend
- Purchase history and monthly trend analytics
- Supabase authentication (signup/login)

## Tech Stack Used
- React + TypeScript
- Tailwind CSS
- shadcn/ui
- Axios
- Vite

## Standard Folder Structure
- `src/components`
- `src/pages`
- `src/context`
- `src/services`
- `src/hooks`
- `src/utils`

## Installation Steps
1. Install dependencies:
```bash
npm install
```
2. Create `.env` and set:
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
- `VITE_API_URL`
- `VITE_SESSION_TIMEOUT_MINUTES` (optional)
3. Run locally:
```bash
npm run dev
```
